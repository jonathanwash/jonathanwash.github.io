Who am I?

Hello, I’m Jonathan Wash. I’m a support engineer for a construction company by day and a technology enthusiast by night.
I’ll be experimenting my experiences on this blog and trying to document everything I do so that other people can hopefully find some help with things… if not only for me to remind me how and why the hell I did things.

The Tech Behind the Blog

This blog is running on GitHub Pages using Hugo as the static generator:

Some fun facts about Hugo:

  • Hugo is a static site engine written in Go.
  • Built in Go
  • Loosely inspired by Jekyll
  • Primarily developed by spf13 on the train while commuting to and from Manhattan.
  • Coded in Vim using spf13-vim
  • Theme is hugo-dusk by Gyorgy Orban

Have questions or suggestions? Feel free to open an issue on GitHub or ask me on Twitter.
